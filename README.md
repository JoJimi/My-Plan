# MyPlan_App
Kotlin을 사용해서 일정 관리 앱을 구현하는 개인 프로젝트를 진행했습니다.   
<br/><br/>

## 👨‍🏫 프로젝트 소개
일정 관리 앱입니다.
<br/>

## 📌 주요기능
```
1. 로그인 기능
2. 회원가입 기능
3. 일정 입력 기능
4. 시간표 기능
5. 스톱워치 기능
6. 사용자 정보
```

## 💻 개발 환경   
- **version** : 'kotlin 1.9.0'
- **IDE** : Android Studio
- **DataBase** : SQLite를 이용한 내부db
